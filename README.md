# Setting Ulrua up on a server!

```bash
sudo git clone https://github.com/oavla/oav.git /var/www/oav

cd /var/www/oav

sh setup.sh
```

---
---

# Setting up Ulrua on Github Codespaces!

```bash
npm install express

sudo npm install pm2 -g

pm2 start index.mjs
```
